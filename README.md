# Mentor Matching: алгоритм подбора менторов

Проект посвящен разработке базового алгоритма подбора менторов в условиях когда у сервиса еще нет данных о взаимодействии пользователей.
В такой ситуации невозможно использовать классические рекомендательные системы, поэтому рекомендации строятся на основе содержимого профилей.

## Идея решения

Алгоритм сопоставляет профили ментора и менти по ключевым признакам и рассчитывает итоговый score как взвешенную сумму.

Основной упор сделан на:
- профессиональную область
- навыки
- цели развития

Дополнительные признаки (формат взаимодействия, индустрия, уровень) используются для уточнения ранжирования.

## Что реализовано

- генерация синтетических профилей менторов и менти  
- content-based алгоритм с rule-based scoring  
- расчет match score  
- ёобъяснение рекомендаций 
- эксперимент по влиянию полноты профиля на качество рекомендаций  

## Эксперимент

Проверяется, как меняется качество рекомендаций при разной заполненности профиля пользователя.

Результаты показывают, что:
- при полной информации модель работает хорошо  
- при частичной — качество снижается  
- при сильно неполном профиле рекомендации становятся менее точными
